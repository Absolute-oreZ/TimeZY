<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the project</p>

```
$ git clone https://github.com/Absolute-oreZ/unFeathered.git
```

<p>2. Navigate to the Project Directory</p>

```
cd timezy
```

<p>3. Run the project</p>

<p>4. Open in Your Browser</p>

```
http://localhost:8080
```
